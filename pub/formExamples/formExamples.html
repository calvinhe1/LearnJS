LearnJS







Add/Edit the title of a learning objective

Write a title on top of the corresponding objective.

            
                //adding or editing titles for given objective number
                learn.editTitle(1,"Learn how to add objectives")
                learn.editTitle(2,"Learn how to view objectives")

Add/Edit the description of a learning objective

The description is the information that is displayed in a convenient manner in relation to a learning objective.

It acts as a summary a user can quickly review.


Add/edit the description of a learning objective, this is the information that will remind users of the details of the learning objective. 

            
                //First argument refers to the objective number to edit, second argument refers to the description itself.
                learn.editDescription(1,"Users can click the press button to add an objective.")

The description can be edited through the form as well - view form/difficulty example for details


Add/Edit the category of a learning objective

Add a search bar for future searches based on categories (will only show learning objectives matching the searched category) and add categories to learning objectives.

            
                //First argument refers to objective number to edit, second argument refers to category itself
                learn.editCategory(1,"Add")
                learn.editCategory(2,"Delete")

The category can be edited through the form as well - view form/difficulty example for details


Changing difficulty level of a learning objective

The purpose of these difficulty levels is to let the user know the amount of time/effort that needs to be invested to successfully understand the corresponding learning objective.


Change the difficulty level of a given learning objective (changes the shape and the color of the border of the objective to the matching difficulty level - see main page for details) 

            
                //Changing difficulty level of first objective from easy to hard
                learn.editDifficulty(1, "hard")

The difficulty level can also be changed if user selects a different difficulty level than the current difficulty in the dropdown of the form and submitting.


Display details of a learning objective

Display all the details about a learning objective in a form (the title, category, and description).

The form can be optionally edited: submitting the form saves any changes and closing discards any changes.


Displaying the form of an objective.

            
                //First argument refers to objective form to display, second argument refers to on or off with 1 or 0 respectively.
                learn.form(1,1)
                
                //learn.form(1,0) to close the form. Must turn off a form before opening another form.

This form can also be triggered if a user right clicks on a learning objective.


Displaying the list of learning objectives

There may be times were users don't want to see the list of learning objectives, so clicking the "hide list" button on the upper top left will hide the list. In reality, the list is entirely removed and then re-added rather than hidden, so it will not occupy space in the document.

Another very convenient feature, is the "move list" button, as it allows the movement of the list of learning objectives to whereever the user's scrolled position is at, allowing the users to interact with the list anywhere in the webpage.
