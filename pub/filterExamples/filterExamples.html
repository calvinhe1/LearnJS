Filter learning objectives based on seached category

After the user clicks on a recommended option or enters, the learning objectives that matched the category show up, as shown here, the "Add" category objectives show up.

Here, objectives can be added/deleted like normal, however if it does not match the current searched category, it will not show up. 

            
                //Filter learning objectives based on "Add"
                learn.filter("Add")

Clicking objectives

Clicking the learning objectives will change color, if red indicating no understnading, yellow indicating some understanding, and green meaning full understanding. This is to help users keep track of their learning process, and how well they receive the learning objectives. 

            
                //Makes the learning objective yellow
                learn.clickObjective(1)

                //Makes the learning objective green if clicked again. If clicked again, will go back to red.
                learn.clickObjective(1)
