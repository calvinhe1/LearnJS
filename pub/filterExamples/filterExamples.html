Filter learning objectives based on searched category

Based on the recommended categories (as shown in the searching example page), if the user clicks on a recommended category or presses the enter key the learning objectives for that searched category show up.

Objectives can be added/deleted like normal, however, if the newly added objective's category does not match the current searched category, it will not show up.


Filtering learning objectives based on the "Add" category. This function holds the same functionality as typing in something in the search bar and entering it. 

            
                //Filter learning objectives based on "Add"
                learn.filter("Add")
