Filter learning objectives based on searched category

After the user clicks on a recommended option (as shown in search example page) or presses the enter key the learning objectives that matches the category show up.

Objectives can be added/deleted like normal, however if the new objective's category it does not match the current searched category, it will not show up.


Search learning objectives based on "Add" category. 

            
                //Filter learning objectives based on "Add"
                learn.filter("Add")
