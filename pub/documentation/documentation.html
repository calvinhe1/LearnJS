LearnJS


Getting Started

Download the LearnJS.js and LearnJS.css file into your working directory, then link the LearnJS.js and LearnJS.css files in your webpage.

Example of basic integration of library into a basic webpage 
            
                <!DOCTYPE html>
                <html lang="en">
                    <head>
                        <meta charset="utf-8">
                        <title>Your title<title>  
                         <link rel="stylesheet" href="LearnJS.css"> 
                        <link rel="stylesheet" href="yourcssfile.css"> 
                        <script defer type="text/javascript" src="LearnJS.js">
                        <script defer type="text/javascript" src="yourjsfile.js">
                        </head>
                        <body>
                        </body>
                        </html>

To get started with using the library, create a LearnJS object in your JS file.

Then, add a list container to hold the learning objectives that will be added to it.


Recommended: Include a hide list button, move list button, and search bar as well to improve the usability of the library (see example pages to see usefulness). 

            
                //Get LearnJS Object
                const learn =  learnJS()

                //Add a list container
                learn.addList()

                //Add search bar for future searching of learning objectives by category
                learn.addSearchBar()

                //Add a hide button to hide/unhide the list of learning objectives
                learn.addHideButton()

                //Add a move button to move the list of learning objectives, so users can view it anywhere
                learn.addMoveButton()

API Object

LearnJS()

This object contains all of the necessary functions for the LearnJS library.


API Methods of LearnJS

addList()

Create a list container to hold learning objectives.


Behaviors

  • Adds an event listener for users to add learning objectives themselves when they click the plus button on the list container. Must submit the popup form to successfully add.

addObjective(difficultyLevel)

Add a learning objective with a difficulty level of difficultyLevel to the list container.


Arguments

  • difficultyLevel: a string that is either "easy", "medium", or "hard".

Behaviors

  • Adds an event listener for users to delete the learning objective when they click the delete button below the learning objective.
  • Adds an event listener for users to display to edit the details (difficulty level, category, title, description) of a learning objective by displaying a form when they right click the learning objective.
  • Adds an event listener for when the user types any key, showing categories that partially or fully matches with what was typed as a dropdown. When the enter key is pressed, the recommended choices are hidden.
  • Adds an event listener for when the user clicks on a recommended category in the dropdown menu in category.
  • Adds an event listener for users to submit the form of any changes of a learning objective by clicking the submit button.
  • Adds an event listener for users to close the form to remove any changes that were added in the current form by clicking the close button.
  • Adds an event listener for users to view the description (key ideas of learning objective) of a learning objective when they mouse over the learning objective (for non empty description).
  • Adds an event listener to hide the description when users mouse out of the learning objective.
  • Adds an event listener for users to change the color of a learning objective to signify current level of understanding to help keep track of their learning progress by clicking on the learning objective.

Return

  • Return the ID of the learning objective (ID starts at 1 and is incremented by 1 for each successive learning objective).

deleteObjective(objectiveNumber)

Delete the learning objective with ID objectiveNumber from the list container.


Arguments

  • objectiveNumber: an integer that is the ID of the objective to be deleted from the list container.

Behaviors

  • Return the ID of the learning objective (ID starts at 1 and is incremented by 1 for each successive learning objective).

editTitle(objectiveNumber, title)

Add/edit the title of a objective number objectiveNumber in the list container.


Arguments

  • objectiveNumber: an integer that is the ID of the objective to be edited.
  • title: the name of a learning objective that is displayed on top of the learning objective figure.

editDescription(objectiveNumber, title)

Add/edit the description of a objective number objectiveNumber in the list container.


Arguments

  • objectiveNumber: an integer that is the ID of the objective to be edited.
  • description: the description of the learning objective that is displayed when user mouses over the learning objective.

editDifficulty(objectiveNumber, difficultyLevel)

Edit the difficulty level of an objective objectiveNumber to difficultyLevel in the list container.


Arguments

  • objectiveNumber: an integer that is the ID of the objective to be edited.
  • difficultyLevel: the difficultyLevel to be applied to the objective.

editCategory(objectiveNumber, category)

Add/edit the category of a objective number objectiveNumber in the list container.


Arguments

  • objectiveNumber: an integer that is the ID of the objective to be edited.
  • category: the category of a learning objective to be used for filtering by category.

changeColor(objectiveNumber, color)

Change the color of an obejctive with objective number objectiveNumber in the list container.


Arguments

  • objectiveNumber: an integer that is the ID of the objective to be changed in color.
  • color: one of red, yellow, or green; the colors to represent understanding of objectiveNumber

form(objectiveNumber, on)

Open/close the form for an objective with objectiveNumber in the list container.


Arguments

  • objectiveNumber: an integer that is the ID of the objective to open the form of
  • on: either 0 or 1, if 1 it opens the form, and if 0 it closes the form. Must close a form before performing other methods (to stay consistent with user interactions)

Behaviors

  • Adds an event listener for users to submit the form of any changes of a learning objective by clicking the submit button.
  • Adds an event listener for users to close the form to remove any changes that were added in the current form by clicking the close button.
  • Add event listener for when the user types any key in the category field, showing categories that partially or fully matches with what was typed as a dropdown. When the enter key is pressed, the recommended choices are hidden.
  • Add event listener for when the user clicks on a recommended category in the dropdown menu in category.
  • Add event listener for when the user selects a different difficulty level, to keep track of the new difficulty level.

showDescription(objectiveNumber, on)

Open the description for an objective with objectiveNumber in the list container.


Arguments

  • objectiveNumber: an integer that is the ID of the objective to open the form of
  • on: either 0 or 1, if 1 it opens the form, and if 0 it closes the form. Must close the popup box before performing other methods (to stay consistent with user interactions). Note: this method is strictly to see what the popup box looks like in the user side.

addSearchBar()

Add a search bar to allow users to search learning objectives by category.


Behaviors

  • Add event listener for when the user types any key, showing categories that partially or fully matches with what was typed as a dropdown. When the enter key is pressed, the search is registered and the learning objectives matching the search shows up. Note: entering an empty space will show all the learning objectives.
  • Add event listener for when the user clicks a recommended category in the dropdown menu, acting as an enter key with the clicked category.

search(category)

Search for a category with the argument category, however don't register in the search, simply show the dropdown menu of recommended categories that match the category.


Arguments

  • category: the category of learning objectives to search for.

filter(category)

Register in the search of category, remove the dropdown menu of recommended categories and show only the learning objectives that matched category


Arguments

  • category: the category of learning objectives to filter for.

addHideButton()

Add a button that allows for the hiding/unhiding of the list container entirely. In reality, it is removed from the DOM and re-added when unhiding and won't conflict with other elements in the webpage.


Behaviors

  • Add an event listener for when the user clicks on the hide list button.

addMoveButton()

Add a button that allows for the movement of the list container entirely. When users are scrolled down and away from the list container that's at the top, the list container can be moved to wherever the user is.


Behaviors

  • Add an event listener for when the user clicks on the move list button.

getCategories()

Get all the categories of the current set of objectives.


Return

  • An array of all the categories of the learning objectives.

getTitles()

Get all the titlesof the current set of objectives.


Return

  • An array of all the titles of the learning objectives.

getDescriptions()

Get all the descriptionsof the current set of objectives.


Return

  • An array of all the descriptionsof the learning objectives.

getDifficulties()

Get all the difficulties of the current set of objectives.


Return

  • An array of all the difficulties of the learning objectives.
