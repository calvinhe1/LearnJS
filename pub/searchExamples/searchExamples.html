LearnJS

Edit the category of a learning objective

Edit the category of a learning objective, this is to group objectives together, so it can be searched easily on the search bar.

            
                //add search bar above the list container to do searches based on category.
                learn.addSearchBar()

                //First argument refers to objective number to edit, second argument refers to category itself
                learn.editCategory(1,"Add")
                learn.editCategory(2,"Delete")

Search objectives based on category

Search the objectives of interest based on the category that was typed in.

Users can type into the search bar as well to trigger the search, and press enter/or click on an existing category to apply the search. Searching empty shows objectives with all categories.

This box can also be triggered if a user mouses over a learning objective. (Only if there exists a description) 

            
                //This mimics a user typing into the search bar, as shown, the search results that match what was entered are shown.
                learn.search("Add")

                //learn.search("") to display all the objectives and remove categories in search bar.
