Search learning objectives based on category

If users are particularly interested in a specific category of objectives to learn, then they can use this feature to view only those objectives. Different capitalizations of the same category are shown, however are considered the same category.


Search the objectives of interest based on category. 

            
                //add search bar above the list container to do searches based on category.
                learn.addSearchBar()

                //This mimics a user typing into the search bar, as shown, the search results match the parameter to the search method
                learn.search("Add")

                //learn.search("") to display all the objectives and remove categories in search bar.

Users can type into the search bar as well to trigger the search.
