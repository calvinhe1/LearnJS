Introduction to Web Programming

HTML

  • HTML stands for hypertext markup language.
  • HTML is very easy to learn and use.
  • HTML is supported in all kinds of browsers, including Internet Edge, Chrome, and Mozilla Firefox.

Some HTML elements

<!Doctype HTML> is information to the browser about what document type to expect.

<html lang="en"> sets the primary language for an HTML document.

<head> is a container for metadata.

<title> is the title of the document, and is shown as the browser's tab name.

<body> contains all information relating to contents of HTML such as headings, paragraphs, images, etc.

<p> represents paragraphs in HTML.

CSS

  • CSS stands for Cascading Styling Sheets.
  • It is used to decorate HTML documents.
  • CSS can do stuff such as alter font, color, size, spacing of HTML elements.

Some CSS examples

p { background-color: yellow; } sets the background-color of all p elements to yellow.

h1 { color: blue; } sets the font-color of all h1 elements to blue.

element { css declarations; } is the general CSS syntax.

JavaScript

  • JavaScript changes behavior of webpages, enabling for interactive pages.
  • HTML, CSS, and JavaScript are essential in web Programming.
  • Majority of websites use JavaScript.
