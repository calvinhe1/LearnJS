LearnJS

Edit the description of a learning objective

Edit the summary/description of a learning objective, this is the set of information a user wants to be reminded of with regards to a learning objective. 

            
                //First argument refers to objective number to edit, second argument refers to the description itself.
                learn.editDescription(1,"Users can click the press button to add an objective.")

Display the description of a learning objective

Display the description/summary about a learning objective in a boxed format.

This box can also be triggered if a user mouses over a learning objective. (Only if there exists a description)

NOTE: The above user functionality will not work in this example page to showcase the popup without a user mousing over the learning objective. 

            
                //First argument refers to the objective description to display, second argument refers to on or off with 1 or 0 respectively.
                learn.showDescription(1,1)
                
                //learn.showDescription(1,0) to close the description. Must turn off this before opening another form/popup.
