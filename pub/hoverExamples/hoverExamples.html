LearnJS







Display the description of a learning objective

Display the description about a learning objective and view the important details to it. 

            
                //First argument refers to the objective description to display, second argument refers to on or off with 1 or 0 respectively.
                learn.showDescription(1,1)
                
                //learn.showDescription(1,0) to close the description. Must turn off this before opening another form/popup.

This box can also be triggered if a user mouses over a learning objective (only if there exists a description).

Note: User functionality will not work in this example page to showcase the popup.
