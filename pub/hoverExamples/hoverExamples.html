LearnJS







Add/Edit the description of a learning objective

The description is the information that is displayed in a convenient manner in relation to a learning objective.

It acts as a summary a user can quickly review.


Add/edit the description of a learning objective, this is the information that will remind users of the details of the learning objective. 

            
                //First argument refers to the objective number to edit, second argument refers to the description itself.
                learn.editDescription(1,"Users can click the press button to add an objective.")

The description can be edited through the form as well - view form/difficulty example for details


Display the description of a learning objective

Display the description about a learning objective and view the important details associated with it. 

            
                //First argument refers to the objective description to display, second argument refers to on or off with 1 or 0 respectively.
                learn.showDescription(1,1)
                
                //learn.showDescription(1,0) to close the description. Must turn off this before opening another form/popup.

This box can also be triggered if a user mouses over a learning objective (only if there exists a description).

Note: The above user functionality will not work in this particular example page (Hover/Description) to showcase the popup.
