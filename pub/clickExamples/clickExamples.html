LearnJS







Add/Edit the category of a learning objective

Add a search bar for future searches based on categories (will only show learning objectives matching the searched category) and add categories to learning objectives.

            
                //add search bar above the list container to do searches based on category.
                learn.addSearchBar()

                //First argument refers to objective number to edit, second argument refers to category itself
                learn.editCategory(1,"Add")
                learn.editCategory(2,"Delete")

The category can be edited through the form as well - view form/difficulty example for details


Search objectives based on category

If users are particularly interested in a specific category of objectives to learn, then they can use this feature to view only those objectives.


Search the objectives of interest based on category. 

            
                //This mimics a user typing into the search bar, as shown, the search results match the parameter to the search method
                learn.search("Add")

                //learn.search("") to display all the objectives and remove categories in search bar.

Users can type into the search bar as well to trigger the search
