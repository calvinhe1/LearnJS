Adding a learning objective

Add learning objectives to a webpage in an aesthetic way and inform users about what they will learn (refer to the main page to understand meaning of shapes and their respective difficulty levels)

Create a list container and add learning objectives to it.

            
                //Get LearnJS library object
                const learn = LearnJS()

                //add list container to hold learning objectives.
                learn.addList()
                
                //Add learning objectives to the list, passing in the difficulty level.
                learn.addObjective("easy")
                learn.addObjective("intermediate")
                learn.addObjective("hard")

Users can add their own learning objectives by clicking the plus button in the bottom of the list container (Note: must submit the form to successfully add)
